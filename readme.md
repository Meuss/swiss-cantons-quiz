## Development

```bash
# Install dependencies
yarn
# Dev server
yarn dev
# Dev server with local netlify functions
netlify dev
# Build for production
yarn build
# Preview production build
yarn preview
# Lint code
yarn lint
# Format code
yarn format
# Type check code
yarn typecheck
```
