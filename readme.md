# List swiss cantons quiz

> A quiz game, playing with React

## :rocket: Quick start

```bash
# Install dependencies
yarn
# Start dev server http://127.0.0.1:5173/
yarn dev
# Build for production
yarn build
# Preview production build
yarn preview
# Lint code
yarn lint
# Format code
yarn format
# Type check code
yarn typecheck
# Run tests
yarn test
yarn test:watch
yarn test:coverage
```
